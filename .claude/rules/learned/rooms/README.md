# Rooms — Rules Organized by Topic

**Purpose**: Topic-based organization (what domain is this about?).

| File | Topic | Description |

| `memory.md` | Memory system | Memory layers, token costs, retrieval patterns |

## Usage

Load rooms when you need topic-based retrieval:
- "What rules apply when writing a hook?" → `hooks.md`
- "What memory patterns have we learned?" → `memory.md`
- "How do we coordinate agents?" → `agents.md`
- "What security rules apply here?" → `security.md`
- "What testing patterns apply?" → `testing.md`
