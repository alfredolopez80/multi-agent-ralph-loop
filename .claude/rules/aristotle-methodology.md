# Aristotle First Principles Methodology

This is the foundational analysis methodology for all tasks in this system.

## Mandatory Process

Before executing any task with complexity >= 4, apply the Aristotle First Principles Deconstructor:

1. **Assumption Autopsy** - Identify inherited assumptions in the problem framing
2. **Irreducible Truths** - List only what survives when all assumptions are removed
3. **Reconstruction from Zero** - Generate 3 approaches using only irreducible truths
4. **Assumption vs Truth Map** - Compare conventional vs first-principles thinking
5. **The Aristotelian Move** - Identify the single highest-leverage action

## Quick Mode (Complexity 1-3)

For simple tasks, apply only:
1. **Assumption Autopsy** - Quick check: are we solving the right problem?
5. **Aristotelian Move** - What is the single best action?

## Integration Point

This methodology is applied during the orchestrator's Step 0 (EVALUATE), BEFORE planning begins. It informs the plan, not the other way around.
